History of files in William Whitaker's website
(http://users.erols.com/whitaker)

This comprehensive file history is gathered mostly from archive.org's scans
of the website.  All the files are in a single directory.  Dates shown here
are the dates the spider read them, not the dates the files were actually
modified.

The Wayback-Machine spider is selective in what it scans.  It prefers to scan
files that have had a lot of updates and makes only infrequent scans of of
other files.  The list below may exclude some updates because of this.

Files not present in an scan I made of the site in March 2013 have entries
which indicate the date by which the files were removed.

This list is supplemented by two other sources:  the mentioned scan of the
site in 2013 and the .zip files containing Windows binaries.  These .zip
files include one or more HTML files (words.htm, wordsdoc.htm, wordsw95.htm)
that were also posted to the website.  In these cases, the date shown is the
date recorded in the .zip file.

Entries for all the files in this list are contained in the web/ directory.
The files are as the were contained on the website--except that
  * Trailing whitespace and ^Z characters are removed from the HTML files
  * Small descriptive placeholder files are substituted these large files:
    - Self-extracting .exe files for Windows and DOS binaries
    - Source-code distribution files wordsall.zip and tools.zip
  * Uppercased files are not stored here when they duplicate other files
